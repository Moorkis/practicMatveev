Перед разворачиванием приложения, необходимо развернуть базу данных через MySql Workbench. Файл для создания базы данных "create_db.sql"
В папке "config" есть файл с конфигурацией базы данных. Если в вашем MySql Workbench отсутствует пароль или он отличен от пароля в файле-конфигураторе, то поменяйте его
